# Linear Quick Start Guide - AdGen AI

## 🚀 Complete This Setup in 15 Minutes

This is your step-by-step checklist to get your Linear workspace fully operational.

---

## ✅ Step 1: Create Teams (5 minutes)

Go to: **[linear.app/settings/teams](https://linear.app/settings/teams)**

Create these 5 teams:

| Team Name | Identifier | Icon | Color |
|-----------|------------|------|-------|
| Engineering | ENG | 💻 | Blue |
| Product | PROD | 🎨 | Purple |
| Marketing & Growth | MKT | 📈 | Green |
| Customer Success | CS | 💬 | Orange |
| Data & Analytics | DATA | 🧠 | Red |

**How to create a team:**
1. Click "New team"
2. Enter team name and identifier
3. Choose icon and color
4. Click "Create team"

---

## ✅ Step 2: Invite Team Members (3 minutes)

Go to: **[linear.app/settings/members](https://linear.app/settings/members)**

**Option A: CSV Upload (Recommended)**
1. Download the template: `/workspace/docs/linear-team-members-template.csv`
2. Replace placeholder emails with actual team member emails
3. Click "Invite members" → "Import from CSV"
4. Upload your edited CSV file

**Option B: Invite Link**
1. Click "Invite members" → "Create invite link"
2. Set permissions (Member or Admin)
3. Share link with your team via Slack/Email

**Option C: Individual Invites**
1. Click "Invite members" → "Email invites"
2. Enter email addresses one by one
3. Assign to appropriate teams

---

## ✅ Step 3: Configure Team Workflows (5 minutes)

For each team, customize the workflow:

### Engineering Team
1. Go to Team Settings → Workflow
2. Set states: `Backlog → Todo → In Progress → In Review → Testing → Done`
3. Add labels: `bug`, `feature`, `technical-debt`, `performance`, `security`
4. Enable GitHub integration

### Product Team
1. Set states: `Research → Design → Ready for Dev → In Development → Shipped`
2. Add labels: `design`, `user-research`, `analytics`, `ux-improvement`

### Marketing & Growth Team
1. Set states: `Ideation → Planning → Execution → Published → Analyzing`
2. Add labels: `content`, `seo`, `competitive-intel`, `agency-program`
3. Set sprint cycle to 1 week

### Customer Success Team
1. Set states: `New → Triaged → In Progress → Resolved → Closed`
2. Add labels: `support`, `bug-report`, `feature-request`, `onboarding`
3. Enable SLA tracking

### Data & Analytics Team
1. Set states: `Research → Prototype → Testing → Production → Monitoring`
2. Add labels: `ml-model`, `fraud-detection`, `attribution`, `data-pipeline`
3. Set sprint cycle to 3 weeks

---

## ✅ Step 4: Set Up Integrations (2 minutes)

### GitHub Integration (Recommended)
1. Go to Settings → Integrations
2. Click "GitHub" → "Install"
3. Authorize Linear to access your repository
4. Select repository: `adgenai` (or your repo name)

**Benefits:**
- Auto-update issues from PR status
- Link commits to Linear issues
- Track code changes per issue

### Slack Integration (Optional)
1. Go to Settings → Integrations
2. Click "Slack" → "Install"
3. Authorize Linear in your Slack workspace
4. Configure notification channels

## 🎯 Best Practices to Start With

### For Team Leads
1. **Create your first issues**: Add 5-10 issues to your team's backlog
2. **Set up labels**: Ensure consistent labeling across issues
3. **Plan first sprint**: Schedule sprint planning meeting
4. **Train team**: Share Linear docs with team members

### For All Members
1. **Set notification preferences**: Settings → Notifications
2. **Install desktop app**: Download from linear.app/download
3. **Learn keyboard shortcuts**: Press `?` in Linear for shortcuts
4. **Connect GitHub**: Link your GitHub account for PR tracking

### For Workspace Admin
1. **Review billing**: Ensure correct plan for team size
2. **Set workspace settings**: Configure defaults and permissions
3. **Monitor adoption**: Check team activity and engagement
4. **Gather feedback**: Ask teams about workflow improvements
